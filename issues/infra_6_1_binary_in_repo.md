# Binario compilado en el repo

El archivo `web-recap` (binario compilado) se encuentra físicamente en la raíz del proyecto. 

Aunque **no** está en el control de versiones de git (ya que `.gitignore` lo excluye correctamente), su presencia puede ser confusa para desarrolladores nuevos. Los artefactos de build deberían generarse idealmente en una carpeta ignorada como `bin/` o `dist/`.
